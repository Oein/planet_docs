# Get started

## 프로젝트 설명

이 프로젝트는 40행성의 데이터팩 컴파일러의 언어를 문서화한 것입니다. 이 문서는 40행성의 데이터팩 컴파일러를 사용하는 방법을 설명합니다.

이 문서에서 작성되는 모든 혜성 코드는 [여기서](https://github.com/Oein/planet_docs/tree/main/example) 볼 수 있습니다.

## 하이라이터

해당 익스텐션은 색만 표시해줄 뿐, 자동완성 기능은 없습니다.

## 사용법

### 1. UI

1. [`compiler.exe`](https://github.com/alexmonkey05/Datapack-Compiler/releases) 를 받아 실행합니다.
2. `.planet` 파일과 데이터팩이 생성될 폴더를 선택합니다.
3. 데이터팩의 이름을 입력합니다. (마인크래프트 namespace와 같습니다)
   1. 기본값은 `pack` 입니다.
   2. 대문자를 입력하면 데이터팩이 올바르게 작동하지 않습니다.
4. "변환하기" 버튼을 클릭합니다.
5. `datapacks` 폴더 안에 생성된 데이터팩과 [`basic.zip`](https://github.com/alexmonkey05/Datapack-Compiler/releases/tag/v2.14)의 내용물을 복사합니다.
6. 마크 안에서 `/reload`를 실행하여 데이터팩을 새로고침합니다.

[혜성 튜토리얼(Youtube Link)](https://youtu.be/vzlmWR5MqCY)

### 2. CLI

1. [`compiler.exe`](https://github.com/alexmonkey05/Datapack-Compiler/releases) 를 받습니다.
2. 혹은 `compiler.py`를 받아 Python(3.12)으로 실행이 가능합니다.

아래와 같이 사용할 수 있습니다.

::::code-group

```bash [Fullname]
compiler.exe --cli --planet ./a.planet --version 1.21 --dist ./world/datapacks --name packpack
```

```bash [Shortname]
compiler.exe --cli -p ./a.planet -v 1.21 -d ./world/datapacks -n packpack
```

::::
