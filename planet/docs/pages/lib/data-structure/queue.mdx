# queue

아래와 같은 코드를 `import`하여 실제 `queue`처럼 사용할 수 있습니다.

```python [queue.planet]
var q = []

def front(){
    return q[0]
}

def pop(){
    del(q[0])
}

def push(var x){
    append(q, x)
}

def size(){
    return len(q)
}

def empty(){
    return len(q) == 0
}

def clear(){
    while(!(empty())){
        pop()
    }
}
```
