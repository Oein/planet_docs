# min/max

`min` 함수는 두 개의 변수 중 작은 값을 반환합니다. `max` 함수는 두 개의 변수 중 큰 값을 반환합니다.

:::info
[GM](/lib/math/gm) 라이브러리를 필요로 합니다.
:::

```python [minmax.planet]
import gm
def min(var x, var y){
    var s = gm.subtract(x,y)
    if(s < 0){
        return x
    }else{
        return y
    }
}

def max(var x, var y){
    var s = gm.subtract(x,y)
    if(s < 0){
        return y
    }else{
        return x
    }
}
```
