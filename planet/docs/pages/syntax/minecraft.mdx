# 마인크래프트 명령어

`/`로 시작하는 경우 마인크래프트 명령어로 인식합니다.

::::code-group

```py[minecraft.planet]
var e = 3
print(e + 1)
/say Hello
```

```py[결과]
4
[Server] Hello
```

::::

`^변수명&`로 입력할경우 마인크래프트 명령어에서 변수를 사용할 수 있습니다.

::::code-group

```py[macro.planet]
var e = 3
/say ^e&
e = e + random()
/say ^e&
```

```py[결과]
[Server] 3
[Server] 3.123456
```
