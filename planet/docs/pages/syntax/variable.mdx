# 변수

## 변수 선언

변수는 값을 저장하는 메모리 공간을 의미합니다. 이는 마인크래프트의 scoreboard와 별개의 것 입니다. 변수를 선언할 때는 `var` 키워드를 사용합니다.

```js [varaiable.planet]
var a;
var b = 2.0;
var c = "Hello, World!";
var d = @p[tag=player];
var e = @p[tag=^b&];
```

여기서 `^b&`는 매크로 기능으로 변수 `b`의 값을 가져옵니다. 이는 혜성의 변수를 마인크래프트에서 사용할 수 있게 해줍니다.

::::code-group

```python [macro.planet]
var a = 1.234
/say ^a&
var b = "가나다"
/say ^b&
```

```python [결과]
1.234
가나다
```
